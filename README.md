# EventPlannerPro (C++ / Qt)

Desktop application developed in C++ using Qt, designed for planning and visualizing event layouts through an interactive graphical interface.

## Features
- Visual arrangement of tables and chairs
- Interactive event layout using Qt Widgets
- Object-oriented and modular design
- Separation between UI, logic, and data components

## Tech Stack
- C++
- Qt (Qt Widgets)
- Visual Studio

## Architecture
- EventPlannerPro – main application window and logic
- ChairItem – graphical and logical representation of chairs
- TableItem – graphical and logical representation of tables
- Modular item-based design for easy extension

## Build & Run
- Requires Qt with Widgets support
- Open `EventPlannerPro.vcxproj` in Visual Studio
- Build and run in Debug or Release mode
